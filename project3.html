Personal Finance Tracker

Advanced dashboard for tracking expenses and visualizing financial health with interactive charts.

Project Overview

Comprehensive dashboard for tracking income and expenses. Users can visualize spending habits through interactive charts built with Recharts, manage categories, and generate monthly reports. The app uses Prisma ORM with PostgreSQL for robust data management.

Key Features

  • Interactive data visualization with Recharts
  • Automated expense categorization
  • Monthly budget tracking and alerts
  • Exportable financial reports (PDF/CSV)
